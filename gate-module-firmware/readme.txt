SUPLA WIFI GATE MODULE

eagle.flash.bin-------->0x00000
eagle.irom0text.bin---->0x40000

PORT SETTINGS:

  RELAY1 GPIO4
  RELAY2 GPIO13
  CFG BUTTON GPIO5

  LED 
      - GREEN GPIO12
      - BLUE GPIO14

  INPUT PORT1 GPIO12
  IMPUT PORT2 GPIO14

  GPIO12,GPIO14 operate in two modes. In and Out (In - INPUT, Out - LED)

  DS18B20 Sensor GPIO2

WROOM PORT SETTINGS:

  RELAY1 GPIO4
  RELAY2 GPIO14
  CFG BUTTON GPIO13

  LED
      - GREEN GPIO5
      - BLUE GPIO12

  INPUT PORT1 GPIO5
  IMPUT PORT2 GPIO12

  GPIO5,GPIO12 operate in two modes. In and Out (In - INPUT, Out - LED)

  DS18B20 Sensor GPIO2
